# Codex Template Install

This document will become the installation guide for the Codex template.

## Intended Install Shape

```text
run setup
  -> install or generate guidance assets
  -> connect skills and overlays
  -> start Codex session
  -> bootstrap
```

## Planned Surfaces

- setup command
- generated overlays
- skills directory wiring
- verification checklist

## Verification Goal

After installation, a new Codex session should:

- read the top-level `AGENTS.md`
- have the expected overlays and skills available
- stay idle until the first actionable request triggers intake

## Status

This is a scaffold.
Concrete commands and generated asset details will be added later.
