# OpenCode Template Install

This document will become the installation guide for the OpenCode template.

## Intended Install Shape

```text
register plugin or config
  -> restart or reload OpenCode
  -> plugin bootstrap activates
  -> skills become discoverable
  -> session starts with bootstrap available
```

## Planned Surfaces

- `opencode.json` plugin registration
- project-level or global config path
- bootstrap text injection
- skills path registration

## Verification Goal

After installation, a new OpenCode session should:

- load the template plugin assets
- discover provider-facing skills
- apply bootstrap behavior before intake begins

## Status

This is a placeholder install guide.
Concrete config snippets and verification steps will be added with the plugin assets.
